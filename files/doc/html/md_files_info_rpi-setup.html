Raspberry Pi

For the current Piloop setup, a Raspberry Pi 3 Model B+ has been used.

OS

Piloop runs on Linux server. No desktop environment is required. It is tested on various debian distros (i.e. diet-pi, Debian server etc)

Components list

To integrate the hardware components to the GPIO pins of the raspberry the following components were used:

  • 2 Breadboards
  • an MCP23017 IO expander (datasheet)
  • an MCP3008 A/DC converter (datasheet)
  • 2-pin buttons
  • 3-pin rotary potentiometers
  • LEDs
  • pull-up resistors
  • wires

Buttons

Apart from the buttons listed below, all other buttons are connected through the IO Expander to the Raspberry Pi:

  • ONOFF
  • PREV_SESSION
  • NEXT_SESSION
  • SAVE_JAM

For more details, see the MCP23017 connections below.

IMPORTANT : The IO expander uses the I2C protocol. You must have enabled the i2c interface on your machine and install i2c-tools.

Potentiometers

Potentiometers are connected to analog to digital circuit of MCP3008. For a detailed overview of the connections, see the MCP3008 connections below.
